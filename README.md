# Traffic_light_pcb
Traffic light using 555 timer and IC4017
It is a 40x42.5mm PCB square shaped two layer.
It doesn't need programming and the speed of the lights can be fixed using pot.
Components:
3 LED (G,Y,R)
6 1N4007 Diodes
Res: 1k, 220 ohm
Cap: 10uF
Battery:9V
Female connectors x2
IC555
IC4017
Potentiometer: 20k
